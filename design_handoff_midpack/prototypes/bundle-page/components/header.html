Component BundleHeader · playground <bundle-header>
A Default — active stage, user is editor
How to use this component
  • Pages that need the header link components/header.css and load components/header.js in <head>.
  • Drop <bundle-header></bundle-header> anywhere in the page.
  • Make it sticky from the host page (it's a plain block element): bundle-header { position: sticky; top: 0; z-index: 30; }.
